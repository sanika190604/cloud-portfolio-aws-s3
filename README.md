# Static Website Hosting using AWS S3

## Project Overview
This project demonstrates hosting a static website using Amazon S3.

The website is built using HTML and CSS and deployed using AWS S3 Static Website Hosting.

## Technologies Used
- AWS S3
- HTML
- CSS
- GitHub

## Architecture
User Browser → AWS S3 Bucket → Static Website Files

## Features
- Static website hosting
- Cloud-based deployment
- Public access configuration

## Deployment Steps
1. Create an AWS S3 bucket
2. Upload website files
3. Enable Static Website Hosting
4. Configure public access
5. Access website through the S3 endpoint
